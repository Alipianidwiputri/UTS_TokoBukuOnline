# Tampilan 9 (LAPORAN PEMESANAN)


<img width="1157" height="860" alt="Screenshot 2025-11-04 164847" src="https://github.com/user-attachments/assets/6f6ba5e8-2570-4f04-ab0a-111cdcda410b" />



**Statistik penjualan:**

- Total transaksi

- Total uang masuk

- Jumlah buku terjual

- Buku paling laris

**Grafik Visual:**

- Grafik garis - penjualan per hari

- Grafik bulat - jenis buku yang terjual

**Detail Transaksi:**

- Tabel semua pesanan

- Nomor pesanan, customer, total, status

**Fitur:**
- Filter waktu - hari ini, minggu ini, bulan ini

- Data real - dari transaksi yang sudah dilakukan

- Export PDF - simpan laporan (simulasi)
