# UTS_TokoBukuOnline

# Penjelasan Halaman Web Aplikasi

**Tampilan 1**
**LOGIN**

Untuk: Masuk ke aplikasi

- Isi email & password
- Klik Login
- Bisa daftar atau lupa password lewat popup

# Tampilan 1 (DASHBOARD)

**Menu utama setelah login:**

Lihat salam sesuai waktu (pagi/siang/sore)

**4 pilihan menu:**

- Stok Buku

- Tracking

- History

- Laporan

# Tampilan 2 (KATALOG/STOK BUKU)

**Fitur:**

- Tambah buku baru - isi form, klik tambah
  
- Lihat daftar buku lengkap (gambar, harga, stok)

- Beli sekarang - langsung checkout

- + Keranjang - tambah ke cart
 
# Tampilan 3 (PROSES BELI)

**Langkah:**

- Klik "Beli Sekarang" atau "+ Keranjang" di stok buku

- Ke halaman checkout - lihat ringkasan pesanan

- Isi data diri seperti Nama, No Handphone dan alamat & Pilih metode pembayaran

- Klik "Bayar Sekarang" - dapat nomor pesanan

**Tampilan 1*
**TRACKING**

**Untuk: Lacak pesanan**

- Masukkan nomor pesanan (contoh: ORD117273)

- Klik Cari - lihat status pengiriman

- Ada progress bar & detail pengiriman

**Tampilan 1**
**HISTORY TRANSAKSI**

**Isi: Riwayat semua pembelian**

- List semua pesanan yang pernah dibuat

- Lihat detail tiap transaksi

- Bisa track pesanan dari sini

**Tampilan 1**
**LAPORAN PEMESANAN**

**Statistik penjualan:**

- Total transaksi

- Total uang masuk

- Jumlah buku terjual

- Buku paling laris

**Grafik Visual:**

- Grafik garis - penjualan per hari

- Grafik bulat - jenis buku yang terjual

**Detail Transaksi:**

- Tabel semua pesanan

- Nomor pesanan, customer, total, status

**Fitur:**
- Filter waktu - hari ini, minggu ini, bulan ini

- Data real - dari transaksi yang sudah dilakukan

- Export PDF - simpan laporan (simulasi)
